# Priority-Queue

A Priority Queue Structure. 

Why write it:

  Priority Queue was written in C++ STL, but it is diffcult to use in Queue module in Python.
  
  This is a easier type.

How to use:

  Function:
  
    Use 'push' to push a value into the queue. Use 'pop' to pop the minimum value out of the queue.

  Property:
  
    Use 'top' to get the minimum value in the queue. Use 'size' to get the size of the queue.

When to use it:

  Priority Queue is a basic data structure. It can be used in some algorithm, like Dijkstra.
